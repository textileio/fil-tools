## pow data get

Get data by cid from the storage profile

### Synopsis

Get data by cid from the storage profile

```
pow data get [cid] [output file path] [flags]
```

### Options

```
  -f, --folder                Indicates that the retrieved Cid is a folder
  -h, --help                  help for get
```
